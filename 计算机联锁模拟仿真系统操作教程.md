# 1 系统概述

本系统是计算机联锁（Computer-Based Interlocking）的模拟仿真平台，运行于 Web 浏览器中。用户可以像车站值班员一样，在站场图上操作信号机与道岔，办理接发列车进路，并观察列车运行过程中区段状态的变化。

> **适用场景：**
>
> - 铁路信号专业教学
> - 联锁逻辑原理演示
> - 故障模拟与应急演练

### 4.2 逐段红光带观察

列车运行过程中，轨道区段会依次经历三种状态，形成“逐段红光带”现象。

```text
列车行驶方向 →

自由（灰） → 锁闭（黄） → 占用（红）
```

或者采用更清晰的示意图：

```text
列车运行方向 →

[灰] ──► [黄] ──► [红] ──► [灰]
 空闲      锁闭      占压      出清
```

假设办理进路 **X→S（通过进路）**，列车经过各区段时状态变化如下：

| 阶段            | IIAG_1 | IIAG_2 | 1AG  | IIG  | 2AG  | 说明                    |
| --------------- | ------ | ------ | ---- | ---- | ---- | ----------------------- |
| 初始            | 灰     | 灰     | 黄   | 黄   | 黄   | 进路已锁闭，列车未到达  |
| 列车进入 IIAG_1 | 红     | 灰     | 黄   | 黄   | 黄   | 接近区段占用            |
| 列车进入 1AG    | 灰     | 红     | 红   | 黄   | 黄   | 首区段占用，IIAG_1 出清 |
| 列车进入 IIG    | 灰     | 灰     | 灰   | 红   | 黄   | 三点检查，1AG 出清      |
| 列车进入 2AG    | 灰     | 灰     | 灰   | 灰   | 红   | IIG 出清                |
| 列车出清        | 灰     | 灰     | 灰   | 灰   | 灰   | 全部区段释放            |

#### 三点检查解锁过程

系统采用“三点检查解锁”机制实现进路逐段释放：

1. 前一区段已经出清；
2. 本区段曾被列车占用；
3. 下一区段已经被列车占用。

满足上述条件后，本区段自动解锁并恢复为空闲状态。

---

### 4.3 列车走完后的自动解锁

当列车完全驶出站场后，系统自动执行进路解锁操作。

解锁后将出现以下变化：

- 所有轨道区段恢复为灰色（空闲状态）；
- 始端信号机自动关闭为红灯；
- 活动进路列表中删除该进路；
- 道岔解除进路锁闭状态；
- 日志区输出提示：

```text
列车已通过，区段逐段出清，进路已解锁
```

此时系统恢复到可重新办理进路的状态。

## 5. 道岔操作

### 5.1 单操道岔

单操用于独立控制道岔的位置转换，可将道岔由定位转换为反位，或由反位转换为定位。

#### 操作步骤

```text

日志显示：

② 点击站场图中的目标道岔（例如 W5）

日志显示：
道岔 5 单操中……（动作延时 0.5s）

③ 等待约 0.5 秒

日志显示：
道岔 5 已转至反位
```

#### 操作结果

- 道岔尖轨方向发生变化；
- 定位显示为绿色；
- 反位显示为蓝色；
- 日志区记录操作过程。

### 5.3 封锁与解封

封锁用于模拟设备施工、维修或故障状态。

#### 封锁操作

```text
① 点击【封锁】

② 点击目标道岔

③ 道岔中心出现红色“封”字标记
```

封锁后：

- 道岔禁止参与任何进路建立；
- 禁止执行单操；
- 系统拒绝所有涉及该道岔的进路。

#### 解封操作

```text
① 点击【解封】

② 点击已封锁道岔

③ 红色“封”字标记消失
```

解封后，道岔恢复正常使用。

- 进路立即解锁；
- 所有相关区段恢复空闲状态；

### 6.2 总人工解锁（已接近进路）

当列车已经接近信号机时，无法执行总取消，只能采用总人工解锁方式。

#### 使用条件

```text
进路已建立

且

接近区段已被占用
```

例如：

```text
IIAG_1 = occupied
```

#### 操作步骤

```text
① 使用 Ctrl + 点击

将接近区段设置为占用状态

② 点击【总人解】

日志显示：
等待选择目标进路

③ 点击始端信号机

系统执行：
- 信号立即关闭
- 启动倒计时
```

#### 倒计时规则

| 进路类型 | 倒计时时间 |
| -------- | ---------- |
| 列车进路 | 180 秒     |
| 调车进路 | 30 秒      |

倒计时期间：

- 信号保持关闭；
- 区段继续锁闭；
- 道岔不可操作；
- 面板实时显示剩余时间。

倒计时结束后：

- 自动释放进路；
- 自动解除锁闭；
- 删除活动进路记录。

### 6.3 倒计时期间的故障安全保护

人工解锁倒计时过程中，系统持续执行安全检查。

#### 进路入侵检测

若检测到列车进入进路内部区段：

```text
任一区段状态变为 occupied
```

系统将：

1. 立即终止人工解锁；
2. 取消倒计时；
3. 转入正常走车解锁模式；
4. 等待列车通过后自动解锁。

#### 列车运行互锁

倒计时期间：

- 禁止恢复暂停列车；
- 禁止重新开放信号；
- 禁止重新办理冲突进路。

该机制确保解锁过程中始终满足故障安全原则（Fail-Safe Principle）。

## 7. 高级调试功能

### 7.1 手动模拟占用（Ctrl + 点击）

系统支持通过 **Ctrl + 点击轨道区段** 的方式，手动切换区段占用状态。该功能主要用于联锁逻辑调试、故障模拟以及验收演示。

#### 操作方法

```text
① 按住键盘 Ctrl 键
（Mac 用户使用 Command 键）

② 点击站场图中的任意轨道区段

③ 区段状态在以下两种状态之间切换：

free（灰色） ↔ occupied（红色）
```

#### 典型应用场景

##### 场景一：模拟列车接近（触发接近锁闭）

```text
用途：
测试总人工解锁功能

操作：

① 建立一条进路
例如：X → IIG

② Ctrl + 点击 IIAG_1

结果：

- IIAG_1 变为红色
- 活动进路标记为“已接近”
- 可执行总人工解锁测试
```

##### 场景二：演示接近区段分级

```text
IIAG_1（红）
= 列车已进入站外区间

IIAG_2（红）
= 列车已接近 X 信号机

1AG（红）
= 列车已越过 X 信号机进入站内
```

暂停功能用于冻结列车运行过程，便于观察联锁状态变化。

#### 操作方法


- 当前占用区段（红光带）

### 7.3 道岔总定与总反

系统支持批量转换全部道岔位置。

| 操作     | 效果                   | 限制                   |
| -------- | ---------------------- | ---------------------- |
| 道岔总定 | 全部可操作道岔转为定位 | 不操作已单锁或封锁道岔 |
| 道岔总反 | 全部可操作道岔转为反位 | 不操作已单锁或封锁道岔 |

执行过程中：

- 已封锁道岔保持不变；
- 已单锁道岔保持不变；
- 活动进路中的锁闭道岔保持不变。

## 8. 故障仿真

### 8.1 信号断丝故障

断丝故障用于模拟室外信号机灯丝损坏场景。

#### 设置断丝故障

```text
① 点击【断丝】

② 点击目标信号机
例如：X

结果：

- 信号灯熄灭
- 显示红色“×”标记
- 日志记录故障信息
```

日志示例：

```text
信号机 X 已设置断丝故障，
灯位熄灭并禁止开放
```

#### 故障影响

当信号机断丝时：

- 禁止开放信号；
- 禁止办理相关进路；
- 系统自动保持禁止状态。

例如：

```text
建立失败：

始端信号机 X 发生断丝故障，
禁止开放信号
```

#### 恢复信号机

```text
① 点击【恢复信号】

② 点击已断丝信号机

结果：

- 断丝标记消失
- 信号恢复正常
```

## 10. 常见问题与技巧

### Q1：点击信号机没有反应？

请检查：

- 当前是否处于“进路操作”模式；
- 信号机是否发生断丝故障；
- 是否存在封锁或锁闭冲突；
- 是否已经选择其他始端信号机。

---

### Q2：总取消或总人工解锁无法使用？

#### 总取消

必须满足：

```text
进路已建立

且

未形成接近锁闭
```

若已形成接近锁闭，则只能使用总人工解锁。

#### 总人工解锁

必须满足：

```text
接近区段已被占用
```

可使用：

```text
Ctrl + 点击接近区段
```

进行模拟。

---

### Q3：无法启动模拟列车？

请确认：

- 已成功建立至少一条进路；
- 当前没有其他列车正在运行；
- 系统未处于人工解锁倒计时状态。

---

### Q4：如何恢复初始状态？

点击：

```text
【上电解锁】
```

系统将：

- 清除活动进路；
- 清除区段占用；
- 清除锁闭状态；
- 恢复道岔位置；
- 恢复信号设备状态。

---

### Q5：常用快捷操作

| 操作             | 快捷方式           |
| ---------------- | ------------------ |
| 模拟区段占用     | Ctrl + 点击区段    |
| 取消始端选择     | 再次点击同一信号机 |
| 查看日志详情     | 滚动日志窗口       |
| 恢复系统初始状态 | 上电解锁           |
