## 攻略社交模块

### 功能描述

#### 1、结构化内容创作

- 多维攻略发布：用户不仅可以发布图文和视频，系统强制/引导输入结构化信息，包括每日路线、真实人均预算、避坑指南
- 动态关联体系：攻略可以直接挂载平台内的“旅游套餐”和“拼团链接”，实现从“种草”到“下单”的无缝衔接
- 多媒体智能处理：集成FFmpeg和OpenCV，自动对用户上传的视频进行转码、抽帧生成封面，对图片进行压缩及水印处理

#### 2、深度互动与结伴

- 互动反馈：提供点赞、收藏、评论功能，通过高频互动数据辅助算法推荐
- 结伴功能：在攻略下方直接开启“求带”、“求拼团”入口，连接有相同出行计划的学生群体

#### 3、智能检索和发现

- 多模态搜索：支持按目的地、预算区间、标签（如：穷游、特种兵、出境）进行精确检索
- 个性化推荐：基于Elasticsearch实现全文检索，并结合用户浏览历史和标签偏好，推送匹配的路线攻略。

#### 4、内容安全与质量保障

- 自动审核流：接入阿里云内容安全API，实时阻断涉黄、涉政及违规广告

- 异步处理机制：利用消息队列确保审核和资源处理不占用主线程，提升发布流程的流畅度

## 技术选型

在内容生产与处理方面，系统通过结构化数据录入和多媒体自动化处理，确保攻略内容的质量与可读性，并通过异步机制提升用户发布体验。

**内容处理与生产组件**

| 组件         | 技术选型       | 核心作用                     | 关键特性                           |
| ------------ | -------------- | ---------------------------- | ---------------------------------- |
| 视频处理引擎 | FFmpeg         | 视频转码、抽帧、压缩         | 支持多格式转换，自动提取攻略封面图 |
| 图像处理库   | OpenCV         | 图片缩放、水印添加、特征提取 | 高性能图像处理，优化移动端加载速度 |
| 内容审核服务 | 阿里云内容安全 | 文本与视觉合规性审查         | 自动识别涉黄、涉政及违规广告内容   |

在内容检索方面，平台采用 Elasticsearch 实现“攻略—路线—预算”的关联检索，满足大学生用户对价格和场景的精准匹配需求。

**搜索引擎组件**

| 组件         | 技术选型      | 核心能力                     | 优化场景                                       |
| ------------ | ------------- | ---------------------------- | ---------------------------------------------- |
| 全文检索引擎 | Elasticsearch | 多维结构化搜索、地理位置检索 | 支持按“预算区间”、“目的地”、“拼团状态”复合筛选 |

在互动统计与社交加速方面，采用 Redis 缓存高频变动的交互数据，利用其原子性操作确保点赞、收藏等计数的准确性与实时性。

**互动与缓存组件**

| 组件       | 技术选型          | 核心功能                   | 特性                                         |
| ---------- | ----------------- | -------------------------- | -------------------------------------------- |
| 高速缓存   | Redis             | 互动数存储、热点攻略缓存   | 支撑高并发点赞/收藏，降低 MySQL I/O 压力     |
| 排行榜引擎 | Redis (SortedSet) | 实时热门攻略、结伴动态榜单 | 基于热度算法自动排序，实现千人千面的动态展示 |

在异步处理与消息通信方面，采用 RocketMQ 作为核心中枢，协调内容发布后的审核、通知及数据同步任务，确保系统的高响应速度。

**消息中间件组件**

| 组件           | 技术选型 | 价值                         | 关键能力                                     |
| -------------- | -------- | ---------------------------- | -------------------------------------------- |
| 分布式消息队列 | RocketMQ | 业务解耦、异步审核、削峰填谷 | 支持事务消息，确保攻略发布与交易链路的一致性 |

在数据存储与资源分发方面，采用混合存储架构。元数据存入关系型数据库，多媒体资源存入对象存储，并通过 CDN 优化学生群体的访问体验。

**存储与分发组件**

| 组件         | 技术选型    | 数据类型                     | 性能优势                                    |
| ------------ | ----------- | ---------------------------- | ------------------------------------------- |
| 关系型数据库 | MySQL       | 攻略元数据、路线详情、预算表 | 结构化存储，支持复杂的“攻略-套餐”关联查询   |
| 对象存储     | MinIO / OSS | 图片、短视频、长文本文件     | 海量存储空间，支持断点续传与高并发读取      |
| 内容分发网络 | CDN         | 静态资源加速                 | 降低跨运营商访问延迟，提升图片/视频打开速度 |

在运维支撑与服务治理方面，系统采用 Spring Cloud Alibaba 生态，通过容器化部署保障社交模块的弹性扩缩容能力。

**治理与部署组件**

| 组件       | 技术选型     | 核心功能           | 关键能力                                   |
| ---------- | ------------ | ------------------ | ------------------------------------------ |
| 服务治理   | Sentinel     | 流量控制、熔断降级 | 保护核心发布接口，防止社交热点流量冲击系统 |
| 链路追踪   | SkyWalking   | 性能监控、故障定位 | 可视化监控攻略发布到展现的全生命周期耗时   |
| 容器化调度 | Docker + ECS | 模块化部署与扩展   | 实现攻略社交模块的独立部署与快速版本迭代   |
